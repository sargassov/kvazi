Сетевой чат "Квази"

Клиент-серверное приложение в виде чата.
Реализовано ввиде двух частей клиентской и серверной.

Клиент. Стек технологий JavaFX.
Реализация двух окон.
 - Главное окно. В зависимости от состояния залогиненного или не залогиненного пользователя
либо показывает окно чата, либо подокно авторизации
 - Окно регистрации нового пользователя. При регистрации необходимо ввести логин, пароль и никнейм.
После регистрации под новыми данными можно войти.

Сервер. Стек технологий Java(Core, Net), JDBC, Lombok, SQLite.
 - Создан сервер с возможностью создавать обработчика каждого подключения в отдельном потоке
 - Создана база данных SQLite, управляется посредством JDBC, в которую по умолчанию
загружены 3 пользователя "asd", "qwe", "zxc". Логины никнеймы и пароли пользователей совпадают.
 - Есть возможность добавить нового пользователя, но хранение его данных будет в режиме inmemory
 - Реализованы возможности отрпавки широковещательных и приватных сообщений.
 - Каждый пользователь может войти только единожды, система не даст зайти второй раз с этим же логином.
